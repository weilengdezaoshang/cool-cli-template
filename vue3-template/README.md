# Cool UI Vue3 模板

这是一个基于Vue3+Vite的项目模板，用于快速启动新项目。

## 特性

- Vue3 组合式API
- Vue Router 路由管理
- Pinia 状态管理
- Vite 构建工具
- ESLint 代码检查

## 快速开始

```bash
# 安装依赖
npm install

# 开发环境启动
npm run dev

# 生产环境构建
npm run build
```

## 项目结构

```
src/
├── assets/       # 静态资源文件
├── components/   # 组件
├── router/       # 路由配置
├── store/        # 状态管理
├── views/        # 页面组件
├── App.vue       # 根组件
└── main.js       # 入口文件
```
